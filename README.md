# Master's of Science in Computer Science - Thesis 

This repository contains all of the files (google collab notebooks, google doc reports, powerpoint presentations) for the Master's of Science in Computer Science Thesis project. These files are provided for the Thesis committee's review. In this study, we investigate the impact of susbsituting real user data with LLM-generated synthetic data for training classification models (RoBERTa, Support Vector Machine, Logistic Regression). The goal is to improve the model's performance (measured by F1, precision, recall) on unseen real user data. For this study, we chose to evaluate the model and LLM-generated synthetic data on two sentiment analysis tasks (a type of classification task), specifically broad-based sentiment analysis and aspect-based sentiment analysis. Broadly speaking, sentiment analysis extracts insights about users' attitudes and preferences which can be used by businesses to improve the customers' experience, improve the brand's reputation, increase consumer retention rate, and overall drive sustainable growth in businesses (which is measured by various key product indicators (KPIs), such as Net Promoter Score, Lifetime Value, Average Revenue per user, etc). Our dataset consists of TripAdvisor reviews from Juneau, Alaska from over 50 locations in Juneau. We chose the locations that are considered "popular" by tourists on TripAdvisor.com. For instance, we scraped reviews from the top ranked locations on the "Things To Do" section, the hotels section, the restaurants section, and the activities section on TripAdvisor.com, generating a robust dataset that provides a comprehensive snapshot of tourists' reactions to different tourist sites in Juneau. We generated the synthetic data using the instruction-tuned Llama3.1 8B (8 billion parameters) in order to augment the minority classes (e.g., the ratings that are underrepresented on TripAdvisor.com). We find that this research can serve as a stepping stone for other studies in other domains (such as education, defense, sustainability) that are broadly exploring how to use synthetic data to create fair and robust AI/ML systems.
