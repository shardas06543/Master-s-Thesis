# Master's of Science in Computer Science - Thesis 

For the description of the README file, we have provided answers to frequently asked questions about this study. Thank you again for reading through this GitHub. We hope it will add to upcoming literature in synthetic data generation, including LLMs and deepfakes. 

What is in this repository? This repository contains all of the files (google collab notebooks, google doc reports, powerpoint presentations) for the Master's of Science in Computer Science Thesis project. These files are provided for the Thesis committee's review. 

What do we investigate in this study? In this study, we investigate the impact of subsituting real user data with LLM-generated synthetic data for training classification models (RoBERTa, Support Vector Machine, Logistic Regression). The goal is to improve the model's performance (measured by F1, precision, recall) on unseen real user data in the underrepresented classes in the imbalanced TripAdvisor dataset. 

What is the hypothesis of this study? The hypothesis is that if we replace a portion of real data with LLM-generated synthetic data in the underrepresented classes only, we will gain an increase in the F1-score, precision, and recall of the underrepresented classes (neutral label and negative label) for broad-based sentiment analysis and aspect-based sentiment analysis. By subsituting real data with LLM-generated synthetic data, we are operating under the assumption that more real data does NOT lead to better models, an assumption that has been studied by prior literature.

Why Sentiment Analysis? For this study, we chose to evaluate the model and LLM-generated synthetic data on two sentiment analysis tasks (a type of classification task): broad-based sentiment analysis and aspect-based sentiment analysis. The goal of sentiment analysis tasks is to extract insights about users' attitudes and preferences which can be used by businesses to improve the customers' experience, improve the brand's reputation, increase the consumer retention rate, and overall drive sustainable growth in businesses (which is measured by various key product indicators (KPIs), such as Net Promoter Score, Lifetime Value, Average Revenue per user, etc). 

What are the contents of the dataset? Our dataset is text-based; It consists of TripAdvisor reviews from Juneau, Alaska from over 50 locations in Juneau. We chose the locations that are considered "popular" by tourists on TripAdvisor.com. For instance, we scraped reviews from the top ranked locations on the "Things To Do" section, the hotels section, the restaurants section, and the activities section on TripAdvisor.com, generating a robust dataset that provides a comprehensive snapshot of tourists' reactions to different tourist sites in Juneau. 

How did we generate the synthetic data? We generated the synthetic data using the instruction-tuned Llama3.1 8B (8 billion parameters) large language model. The role of the LLM-generated synthetic data is to augment the minority classes (e.g., the ratings that are underrepresented on TripAdvisor.com) in order to increase the diversity of the dataset. We find that this research can serve as a stepping stone for other studies in other domains (such as education, defense, sustainability) that are broadly exploring how to use synthetic data to create fair and robust AI/ML systems.
