# Master's Thesis

This repository contains all of the files (google collab notebooks, google doc reports, powerpoint presentations) for the Master's in Science Thesis, for the committee's review. In this study, we investigate the impact of using synthetic data for training classification models. Our dataset consists of TripAdvisor reviews from Juneau, Alaska in 50+ locations. We generated the synthetic data in order to augment the minority classes (e.g., the ratings that are underrepresented on TripAdvisor.com).
