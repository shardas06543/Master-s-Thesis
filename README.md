# Master's of Science in Computer Science - Thesis 

This repository contains all of the files (google collab notebooks, google doc reports, powerpoint presentations) for the Master's in Science of Computer Science Thesis. These files are provided for the Thesis committee's review. In this study, we investigate the impact of using synthetic data for training classification models (RoBERTa, Support Vector Machine, Logistic Regression). For this study, we chose to hone in on sentiment analysis tasks, specifically broad-based sentiment analysis and aspect-based sentiment analysis. Our dataset consists of TripAdvisor reviews from Juneau, Alaska from over 50 locations in Juneau. We chose the locations that are considered "popular" by tourists on TripAdvisor.com. For instance, we scraped reviews from the "Things To Do" section, the hotels section, the restaurants section, and the activities section on TripAdvisor.com, generating a robust dataset that provides a comprehensive snapshot of tourists' reactions to different tourist sites in Juneau. We generated the synthetic data using the instruction-tuned Llama3.1 8B (8 billion parameters) in order to augment the minority classes (e.g., the ratings that are underrepresented on TripAdvisor.com). We find that this research can serve as a stepping stone for other studies in other domains (such as education, defense, sustainability) that are broadly exploring how to use synthetic data to create fair and robust AI/ML systems.
