# Master's Thesis

This repository contains all of the files (google collab notebooks, google doc reports, powerpoint presentations) for the Master's in Science Thesis, for the committee's review. In this study, we investigate the impact of using synthetic data for training classification models. Our dataset consists of TripAdvisor reviews from Juneau, Alaska from over 50 locations in Juneau. We chose the locations that are considered "popular" by tourists on TripAdvisor.com. For instance, we scraped reviews from the "Things To Do" section, the hotels section, the resturants section, and the activities section, generating a robust dataset that provides a comprehensive understanding of tourists' reactions to different tourist sites in Juneau. We generated the synthetic data in order to augment the minority classes (e.g., the ratings that are underrepresented on TripAdvisor.com).
